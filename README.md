# Fear Ladder

A simple application for tracking and managing fear ladders.

## Features

- Custom fear ladder entries
- Progress tracking
- Exposure therapy support

## Technologies

- React
- Vite
- Tailwind CSS
- Shadcn UI
- Supabase

## Setup

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start development server:
   ```sh
   npm run dev
   ```
